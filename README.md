# Connect Four — MCTS + Neural Network

A C++ implementation of Connect Four with an AlphaZero-style AI player and a graphical interface.

## Features

- Connect Four game engine (6×7 board)
- AI player based on Monte Carlo Tree Search (MCTS)
- Dual-head neural network: policy head + value head
- Self-play training loop
- GUI built with [raylib](https://www.raylib.com)

## Dependencies

- C++17
- [tiny-dnn](https://github.com/tiny-dnn/tiny-dnn)
- [raylib](https://www.raylib.com)
